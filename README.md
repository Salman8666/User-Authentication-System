# User-Authentication-System
It is a secure web-based user authentication and survey submission system developed using Python Flask, MySQL, HTML, and CSS. The application allows users to create an account, log in securely, complete a survey form, submit their responses, and log out. User authentication ensures that only registered users can access and submit the form.
Features
User registration (Sign Up)
Secure user login authentication
Session management using Flask
Protected survey form accessible only after login
Form submission with data stored in MySQL
User logout functionality
Responsive and user-friendly interface
Tech Stack
Frontend: HTML5, CSS3
Backend: Python, Flask
Database: MySQL
Development Tools: VS Code
Project Workflow
User creates a new account.
User logs in with valid credentials.
Authenticated user accesses the survey form.
User fills in and submits the form.
Survey responses are stored securely in the MySQL database.
User logs out, ending the session.
Learning Outcomes

This project demonstrates:

User authentication and authorization
Flask routing and session management
CRUD operations with MySQL
Form handling and validation
Frontend and backend integration
Database connectivity using Python

This project was developed as part of learning full-stack web development using the Flask framework and MySQL database.
